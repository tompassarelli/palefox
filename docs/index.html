Palefox

Firefox, unfenced

Install GitHub

Features

Tree-Style Tabs

Custom tab panel with deep nesting, groups, drag-and-drop reordering, and multi-select.

Vim Keybindings

Navigate, refile, search, and manipulate tabs without leaving the keyboard. Ex commands included.

Compact Mode

Sidebar autohides off-screen, slides in on hover with spring animation. Ported from Zen Browser's state machine.

Theme Support

Uses Firefox's native CSS variables. Works with any theme — system, extension, or custom.

Quick Install

Requires Firefox with vertical tabs enabled.

macOS / Linux curl -fsSL https://raw.githubusercontent.com/tompassarelli/palefox/main/install.sh -o /tmp/palefox-install.sh && bash /tmp/palefox-install.sh
Windows (PowerShell) irm https://raw.githubusercontent.com/tompassarelli/palefox/main/install.ps1 | iex

LibreWolf? Add --librewolf to either command. Installer pulls from the latest tagged release. See the full install guide.
